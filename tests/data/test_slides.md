# Simply Supported Beam

For a uniformly distributed load:

$$M_{max} = \frac{wL^2}{8}$$

Where:

- $w$ = distributed load (kN/m)
- $L$ = span length (m)
- $M_{max}$ = maximum bending moment (kNm)

::: notes
:: Reference: Chapter 5, Section 3
:: Author: Make sure to explain each variable clearly

[MIN 12s]

For a simply supported beam carrying a uniformly distributed load, the maximum bending moment occurs at the center of the span.
[PAUSE 1s]
The formula is w L squared divided by eight.
[PAUSE 1s]
Here, w represents the distributed load in kilonewtons per meter, L is the span length in meters, and M max is the maximum bending moment in kilonewton meters.
This fundamental relationship is essential for designing beams in structural engineering.
